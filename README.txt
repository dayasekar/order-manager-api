Pre-requisites
    Your favourite IDE needs to be enabled with lombok plugin for the source code to display without any
    compilation errors.

Running the application
    Please execute the following command
        mvn spring-boot:run
    OR
    Please execute the following command
        mvn clean package
    cd to target directory and execute
        java -jar order-manager-api-1.0.0-SNAPSHOT.jar

Testing the API
    You can test the order manager API using the swagger URL at
        http://localhost:8080/swagger-ui.html
